# Faith Journal App Icons

This folder contains all the app icons for the Faith Journal app, organized for easy access and management.

## 📁 Folder Structure

### Faith_Journal_Icon_Set/
Complete organized icon set with proper .imageset folders for Xcode.

**Contents:**
- AppIcon-1024x1024.imageset (App Store icon)
- AppIcon-20x20.imageset (iPhone Notification)
- AppIcon-29x29.imageset (iPhone Settings)  
- AppIcon-40x40.imageset (iPhone Spotlight)
- AppIcon-60x60.imageset (iPhone App)
- AppIcon-76x76.imageset (iPad)
- AppIcon-83.5x83.5.imageset (iPad Pro)
- And more...

### Key Files:
- **AppIcon-1024x1024.png** - Main App Store icon (1.3MB)

## 🎯 Usage

### For Xcode Asset Catalog:
The main App Store icon is already properly placed in:
`Assets.xcassets/AppIcon.appiconset/AppIcon-1024x1024.png`

Xcode will automatically generate all smaller icon sizes from this 1024x1024 master icon.

### For Manual Editing:
Use the files in `Faith_Journal_Icon_Set/` if you need to manually adjust specific icon sizes.

## 📋 Icon Specifications

- **Format**: PNG with transparency
- **Color Space**: sRGB  
- **App Store**: 1024×1024px minimum
- **File Size**: Main icon is ~1.3MB
